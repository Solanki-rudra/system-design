# Fundamentals

Welcome to the **Fundamentals** section! This directory contains the foundational concepts you need to understand before diving into large-scale system architecture. 

## 📂 File Directory

- [`what-is-system.md`](./what-is-system.md) - Explains what a system is, and the basics of distributed computing environments.
- [`cap-theorem.md`](./cap-theorem.md) - Deep dive into the CAP Theorem (Consistency, Availability, Partition Tolerance) including practical use-cases and scenarios.

*(Note: As you study more fundamentals like Load Balancing or Caching, add them here!)*

## 💡 Key Insights
- You cannot understand massive system architectures (like Netflix or Uber) without first grasping these foundational building blocks.
- Real-world distributed systems are all about managing the tradeoffs between these basic components.

## 🚀 How to Use
1. Click the links above to read the files sequentially.
2. Ensure you comfortably understand the CAP theorem tradeoff before progressing to the `core-concepts` folder.
