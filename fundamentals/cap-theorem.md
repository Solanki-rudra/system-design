# CAP Theorem

## Definition
A distributed system can only guarantee 2 out of:
- Consistency
- Availability
- Partition Tolerance

## Reality
Partition tolerance is mandatory → so trade-off is:
- Consistency vs Availability

## Use Cases
- Banking → Consistency
- Social Media → Availability

## Key Insight
Perfect system doesn't exist → only tradeoffs

## Interview Tip
Always ask: "What matters more here?"
