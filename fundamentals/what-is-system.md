# What is a System?

## Definition
A system is a collection of components that work together to solve a problem.

## Core Characteristics
- Input
- Output
- Boundaries

## Why Boundaries Matter
- Prevent scope creep
- Keep problem finite
- Help focus on relevant details

## Key Insight
Same components ≠ same system  
→ Behavior depends on interactions + implementation details

## Interview Angle
- Always define system boundaries early
- Clarify what's inside vs outside
